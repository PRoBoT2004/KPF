## 🎨 Design System

The portfolio uses a comprehensive design system ensuring consistency across all components:

### Typography Scale
- **H1**: 5xl-7xl (Hero headings)
- **H2**: 4xl-6xl (Section headings)
- **H3**: 3xl-4xl (Subsection headings)
- **H4**: 2xl-3xl (Card headings)
- **H5**: xl-2xl (Small headings)
- **Body**: base-lg (Main content)
- **Small**: sm-base (Secondary content)

### Color Palette
- **Primary**: Orange (#f97316) - Main brand color
- **Secondary**: Slate (#64748b) - Supporting elements
- **Accent**: Blue (#3b82f6) - Interactive elements
- **Neutral**: Gray scale (#f8fafc to #0f172a) - Text and backgrounds

### Component Styles
- **Buttons**: Consistent button styles with hover effects
- **Cards**: Glass-morphism design with backdrop blur
- **Sections**: Standardized spacing and layout patterns
- **Animations**: Smooth transitions and micro-interactions

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd KPF-master

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
